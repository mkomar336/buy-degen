## 5. 存储您的 Degen

虽然交易所提供存储功能，但为了更高的安全性，建议将您的 Degen 转移到加密钱包中。

### 钱包类型
1. **软件钱包（热钱包）：** 连接互联网，使用方便但安全性较低。  
2. **硬件钱包（冷钱包）：** 离线存储，更安全，推荐 [Ledger](https://shop.ledger.com/)。

### 种子短语保护
硬件钱包会生成种子短语，建议将其存储在防火、防水的金属板上以防丢失。
